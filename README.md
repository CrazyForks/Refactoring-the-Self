### 为什么写《底层重构》

人生和软件系统其实很像。都会积累技术债，都会经历失控，也都需要不断重构。

把这些经历写下来，既是对自己的交代，也是想把那些走过的弯路、吃过的苦、重新站起来的过程，留给后来的人看一看。
